Pour faire fonctionner le webservice du sandbox, il faut : 


  1- Créer un dossier appelé : C:\HomeSandBox 
     c'est dans ce dossier que la Sandbox créer des domains et ou les fichiers exe des exercices sont transférés.


  2- Créer un fichier text des permissions. le fichier doit avoir au minimum Execute soit : 

      write, read,  execute , createfile
      
     et chaque permission doit être écrite dans une ligne (sans virgule ou autre), voir un exemple de fichier Prmission.txt 


  3- Pour avoir les résultats de l'exécution dans un fichier text, rajouter ces lignes de code dans le main de l'exécutable crée
       
       Le "PathDuFichierResult" peut être comme suit : @"C:\HomeSandBox\TEST.exe.Result.txt"

       StreamWriter result = new StreamWriter(PathDuFichierResult);
       Console.SetOut(result);

       ////////////////////////////////////////////////////////////////////////////////        
       ////////////// Ici le code de l'exercice! //////////////////////////////////////
       ////////////////////////////////////////////////////////////////////////////////

       result.Flush();
       result.Close();
